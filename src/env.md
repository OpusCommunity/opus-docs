# 動作環境

## OS

Opus は WebView2 を使用しています。WebView2 は自動的に更新されるため、すべての Windows ターゲットが保証されます。

###### 対応バージョン

| opus バージョン | 対応 OS   | バージョン |
| --------------- | --------- | ---------- |
| 0.1             | Windows10 | 2004 以降  |
|                 | Windows11 | 21H2 以降  |
|                 | MacOS     | 19 以降    |
